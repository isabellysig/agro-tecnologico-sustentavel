# 🌱​ agro-tecnologico-sustentavel
Sobre o uso da tecnologia na agricultura e sua função na redução do desperdicio de recursos naturais.
---
# 🍃​ objetivo do projeto
O projeto tem como seu principal foco que as pessoas possam observar de modo consciente o uso e a preservação do ambiente e dos recursos utilizados na agricultura, como água, solo e os minerais, e como inovações podem ajudar a manter o equilibrio natural

---

# recursos tecnicos
## acesibilidade
Narrador de voz;
Tradução;
Aumento e diminuição de fontes.

## tecnologias
Estrutura: html
Estilização: css
Interação: java script

---

# publico alvo
Esse site tem o intuito de ser agradavel a qualquer faixa de idade, mas principalmentes pessoas que tem laços com agricultura de grande porte para sensibilizar suas escolhas

---

# impacto socioambiental
O projeto contribui para sensibiação sobre o uso consiente de recursos muitas vezes não renovavei, inventivando praticas sustentaveis e contando em detalhes seus maiores beneficios, compreendendoa importancia desse ato simples que algumas vezes é deixado de lado
